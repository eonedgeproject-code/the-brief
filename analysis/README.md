# analysis/

Research notes that draw on the captures in [../entries/](../entries/). These are *interpretation*,
not record: a note may reason, compare, and speculate, but every factual claim should trace back to
a filed entry or a clearly-labelled source, and a note that leans on a low-confidence entry inherits
that weakness and says so.

An analysis is not a capture. Nothing here is a verbatim system prompt — for that, see the entries.
